# Florian Rischer - UX/UI Portfolio

A modern, interactive portfolio website built with React, TypeScript, and Vite showcasing UX/UI design projects with smooth animations and interactive project detail pages.

## Getting Started

### Installation

Install the project dependencies using npm:

```bash
npm install
```

### Development

Start the development server with hot module replacement:

```bash
npm run dev
```

The portfolio will be available at `http://localhost:5173` (or another port if 5173 is in use).

## Build

To create a production build:

```bash
npm run build
```

## Technologies

- **React 19** - UI framework
- **TypeScript 5.9** - Type-safe JavaScript
- **Vite 7.2** - Lightning-fast build tool
- **React Router 7.11** - Client-side routing
- **GSAP** - Advanced animations
- **ESLint + Husky** - Code quality and pre-commit hooks

## Features

- Smooth page transitions with GSAP animations
- Interactive project detail pages with screen carousels
- Responsive design with CSS variables
- Zoom functionality on project mockups
- Skill ratings and filtering
- Contact form with validation
- Social media integration
